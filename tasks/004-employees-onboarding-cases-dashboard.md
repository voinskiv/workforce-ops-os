# Task 004 — First Vertical Slice

## Goal

Build the first usable vertical slice.

## Scope

Implement:

1. Employee list
2. Employee detail page
3. Onboarding pipeline table
4. Operational case board
5. Basic dashboard

## Acceptance Criteria

Dashboard shows:

- total employees
- active employees
- planned employees
- no-shows
- open onboarding items
- open cases
- open time issues

Employee list supports:

- status filter
- partner filter
- customer filter
- open detail page

Onboarding supports:

- advancing status
- marking no-show
- creating a no-show case

Cases support:

- list
- create
- update status
- filter by status/type/priority
